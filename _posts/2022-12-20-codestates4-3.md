## SSH (Secure SHell)
- 보안이 강화된 shell 접속
- CLI 환경에서 다른 PC에 접속하거나 요청할 때 사용
- 비대칭키를 이용해 사용자 인증

## SSH 키 생성
- ```
   ssh-keygen //기본 암호화 알고리즘 방식으로 키 생성
   ssh-keygen -t ed25519 -C "깃허브이메일" //표준 암호화 알고리즘 ed25519 방식으로 명시적 생성
  ```
  - 기존엔 rsa 방식을 사용함
- Enter로 넘김
- ~/.ssh/ 경로에 id_rsa/ed25519 와 id_rsa/ed25519.pub 생성 → ssh 키 페어
  - id_rsa/ed25519.pub : 공개키(Public Key) 
  - id_rsa/ed25519 : 개인키(Private Key) 또는 비밀키(Secret Key)
   
## 공개키(Public Key) 복사
- ```
   cat ~/.ssh/id_rsa.pub
   cat ~/.ssh/id_ed25519.pub
  ```

## Github에 공개키 등록
- Github - Settings - SSH and GPG keys - New SSH Key(SSH Keys)
- SSH 공개키를 구분할 수 있는 Title을 작성하고 공개키 붙여넣기
