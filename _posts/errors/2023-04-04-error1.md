Githun Actions 워크플로우 생성 중 발생한 에러이다. 
```
git update-index --chmod=+x gradlew
```
build 시 발생하는 권한 문제로, 위 명령어를 통해 그 권한을 부여해준다. 

<br>

또는 .yml 파일에 아래의 소스를 빌드 전에 추가해준다. 

```
- name: Run chmod to make gradlew executable
   run: chmod +x ./gradlew
```
